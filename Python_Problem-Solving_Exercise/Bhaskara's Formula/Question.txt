Read 3 floating-point numbers. After, print the roots of Bhaskara’s formula. If it's impossible to calculate the roots because a division by zero or a square root of a negative number, presents the message “Impossible calculation”.

Input
Read 3 floating-point numbers (double) A, B, and C.

Output
Print the result with 5 digits after the decimal point or the message if it is impossible to calculate.

Input Samples:
10.0 20.1 5.1

Output Samples:
R1 = -0.29788
R2 = -1.71212
